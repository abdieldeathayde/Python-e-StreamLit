# Python + Streamlit 🚀

Projeto desenvolvido durante os estudos da Hashtag Programação utilizando **Python** e **Streamlit** para criação de aplicações web interativas de forma rápida e prática. O objetivo deste repositório é praticar conceitos de desenvolvimento backend, manipulação de dados e construção de interfaces web utilizando apenas Python.

## 📌 Tecnologias Utilizadas

* Python 3
* Streamlit
* Pandas
* Git & GitHub

## 📂 Estrutura do Projeto

```bash
Python-e-StreamLit/
│
├── app.py
├── requirements.txt
├── assets/
├── pages/
└── README.md
```

## 🚀 Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/abdieldeathayde/Python-e-StreamLit.git
```

### 2️⃣ Acesse a pasta do projeto

```bash
cd Python-e-StreamLit
```

### 3️⃣ Crie um ambiente virtual (Opcional, mas recomendado)

#### Linux / MacOS

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### 4️⃣ Instale as dependências

```bash
pip install -r requirements.txt
```

### 5️⃣ Execute o projeto

```bash
streamlit run app.py
```

Após executar o comando, o Streamlit abrirá automaticamente no navegador:

```bash
http://localhost:8501
```

## 📖 O que é Streamlit?

O Streamlit é um framework open source em Python utilizado para criar aplicações web interativas voltadas para ciência de dados, dashboards e aplicações web rapidamente, sem necessidade de frontend complexo. ([GitHub][1])
