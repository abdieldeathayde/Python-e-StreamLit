# Python + Streamlit + Groq 🚀

Aplicação web desenvolvida com **Python**, **Streamlit** e integração com a API da **Groq**, permitindo interações inteligentes através de uma interface moderna e rápida.

O projeto foi criado com foco em aprendizado prático de:

* Desenvolvimento backend com Python
* Integração com APIs de IA
* Criação de aplicações web com Streamlit
* Deploy de aplicações em nuvem

---

## 📌 Tecnologias Utilizadas

* Python 3
* Streamlit
* Groq API
* Pandas
* Git & GitHub

---

## 🤖 Sobre a Groq

A **Groq** é uma plataforma de inferência de IA focada em alta performance e baixa latência, oferecendo acesso rápido a modelos de linguagem modernos através de APIs simples e eficientes.

🔗 Documentação Oficial:  
https://console.groq.com/docs
